# 📈 Блок 2: Прогнозирование посещаемости

Прогноз количества пользователей сервиса на 12 месяцев.

## 📋 Задача

**Дано:** Исторические данные 2015-2025 (123 месяца)  
**Цель:** Спрогнозировать посещаемость на апрель 2025 - март 2026

## 📊 Данные

**Файл:** `data/newborns_stats.csv`

**Переменные:**
- `year`, `month` — период
- `target` — количество пользователей (целевая переменная)
- `cat1` — категория (0=зима/весна, 1=лето, 2=осень)

**Статистика:**
- Среднее: ~11,000 пользователей/месяц
- Максимум: 14,857 (июль 2015)
- Минимум: 6,611 (март 2025)
- **Тренд:** Снижение посещаемости

## 🔮 Результат

**Прогноз на 12 месяцев:**
- Средний: ~8,800 пользователей/месяц
- Летние месяцы: ~9,500-10,000
- Зимние месяцы: ~8,000-8,500
- **Изменение:** Продолжение нисходящего тренда (-15-20%)
